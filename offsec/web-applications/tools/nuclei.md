### Installation

```csharp
git clone https://github.com/projectdiscovery/nuclei.git; cd nuclei/v2/cmd/nuclei/; go build; mv nuclei /usr/local/bin/; nuclei -h

```

###  Update Templates

```csharp
nuclei -update-templates
```
